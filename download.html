Installation & First Launch

Getting started with BLOODSPIRE is now simpler than ever thanks to the official installer.

What the Installer Does

  • Automatically place all game files into the C:\Bloodspire directory.
  • Install a portable version of Python (no manual Python installation required).
  • Install a client version of Tailscale (the secure VPN tunneling software needed for league play).
  • Create a convenient Bloodspire shortcut on your Desktop.

Step-by-Step Installation

  1. Download and run the installer from the button above. Follow the on-screen prompts; the process is fully automatic.
  2. After installation, double-click the Bloodspire shortcut on your Desktop. This launches the game client.
  3. A command window will open in the background (do not close it while playing), and your default web browser should automatically open the game at http://localhost:8765.
  4. If the browser does not open automatically, manually navigate to: http://localhost:8765
Note: Always launch the game using the Desktop shortcut. The command window must remain open during play.

Tailscale Setup (Required for League Play)

BLOODSPIRE league play uses Tailscale to create a secure private network between you and the Arena Master’s server.

  1. Create a free Tailscale account: Go to tailscale.com/download and sign up for a free account.
  2. Contact the Arena Master: Email bkstafford1971@gmail.com with your Manager Name and the email address used for Tailscale.
  3. Join the League Network: The Arena Master will send a secure invitation link. Click it while Tailscale is running and accept to join the private league network.

Connecting to the League Server

Once Tailscale shows your device is "Connected" to the league network:

  1. In the BLOODSPIRE client, go to Action → League Settings…
  2. The League Server URL field should be automatically filled in as:
    http://100.114.138.61:8766
  3. If it is not, manually enter that exact URL.
  4. Enter your manager name and password and click Register & Save.
Important Progress Notes:
  • Your progress saves automatically in the saves folder inside C:\Bloodspire.
  • You must be connected via Tailscale to participate in official matches.
  • Upload your teams before each deadline using Action → Upload.
